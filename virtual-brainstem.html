How it works ▶

Built-in ManageMemory and ContextMemory agents persist to a localStorage-backed virtual filesystem. Drop any agent.py onto the chat to hot-load it instantly. Sign in with GitHub to chat through GitHub Models — your Copilot subscription is the engine.
