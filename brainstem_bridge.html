🧠🌉 Brainstem Bridge

Fronts a local brainstem into the RAPP neighborhood — so another browser, anywhere, can drive this computer's brainstem over WebRTC. Speaks rapp-neighborhood-protocol/1.0 (the same twin-chat envelope as the vBrainstem), so a remote peer can't tell it's bridged.

Every request is gated by a one-time session token minted below — share the operator link only with yourself (same GitHub account). Closing this tab ends the session, exactly like stopping the brainstem server.
